How to use this app:

1. write in the console following: "npm init"
2. then configure all three of these by following instructuions left inside the files:
    docker-compose.yml
    .env
    user-table.sql

3. write following in the console: "docker compose up -d"

4. to initialize the server simply run "npm run dev" in the console.
    if such command doesnt exist, simply add *"dev": "nodemon src/index.js"* in the scripts section of the package.json

5. ENJOY!
